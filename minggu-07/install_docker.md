Installasi Docker pada Windows
----------
> Step 1: Download installer terlebih dahulu

![1](./img/install-01.jpg)


> Step 2: Mulai Install Docker di Windows

![2](./img/install-02.jpg)


> Step 3: Konfigurasi. Kita perlu konfigurasi agar Docker menggunakan WSL 2

![3](./img/install-03.jpg)


> Step 4: Proses Instalasi

![4](./img/install-04.jpg)


> Step 5: Instalasi Selesai. Klik Close and restart untuk restart komputer.

![5](./img/install-05.jpg)


> Step 6: Setelah restart, Docker akan terbuka otomatis dan kita akan melihat jendela seperti ini. Centang I accept the terms, kemudian klik Accept.

![6](./img/install-06.jpg)


> Step 7: Install WSL 2

![7](./img/install-07.jpg)

Klik link pada jendela dialog untuk download WLS 2.

![8](./img/install-08.jpg)

Setelah kita download, buka installer wsl_update_x64.exe

![9](./img/install-09.jpg)

Klik Next untuk memulai instalasi.

![10](./img/install-10.jpg)

Tunggu sampai prosesnya selesai, setelah itu klik Finish.

![11](./img/install-11.jpg)

Setelah itu, barulah kita bisa restart.

![12](./img/install-12.jpg)
